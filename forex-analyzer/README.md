# Forex Analyzer

Приложение для анализа валютных пар на форексе и отправки сигналов в Telegram.

## Описание

Приложение проверяет валютные пары за 10 минут до окончания каждого часа и отправляет сигналы:

- **BUY (Покупка)**: если последняя дневная свеча зеленая И последние 2 часовых свечи зеленые
- **SELL (Продажа)**: если последняя дневная свеча красная И последние 2 часовых свечи красные

## Технологии

- Next.js 14
- TypeScript
- React
- Node.js
- Telegram Bot API
- Finnhub API (данные Forex)

## Установка

```bash
cd forex-analyzer
npm install
```

## Настройка

1. Создайте файл `.env.local` в корне проекта:

```env
TELEGRAM_BOT_TOKEN=your_bot_token_here
TELEGRAM_CHAT_ID=your_chat_id_here
FINNHUB_API_KEY=your_finnhub_api_key_here
```

2. Получите токен бота:
   - Откройте @BotFather в Telegram
   - Отправьте команду `/newbot`
   - Следуйте инструкциям

3. Получите Chat ID:
   - Добавьте бота в чат или канал
   - Отправьте сообщение
   - Используйте @userinfobot или аналогичный для получения ID

4. Получите бесплатный API ключ Finnhub:
   - Зарегистрируйтесь на https://finnhub.io/
   - Получите бесплатный API ключ (60 запросов/минуту)
   - Добавьте его в `.env.local`

## Использование

### Запуск веб-интерфейса

```bash
npm run dev
```

Откройте http://localhost:3000 в браузере.

### Ручной запуск анализа

```bash
npm run analyze
```

### Автоматический запуск (cron)

Добавьте в crontab (`crontab -e`):

```bash
# Запуск каждые 10 минут до окончания часа (в :50)
50 * * * * cd /path/to/forex-analyzer && /usr/bin/npm run analyze >> /var/log/forex-analyzer.log 2>&1
```

### API endpoint

GET запрос на `/api/analyze`:

```bash
curl http://localhost:3000/api/analyze
```

## Логика работы

1. Скрипт получает данные свечей с Finnhub API (бесплатный Forex API)
2. Анализирует последнюю дневную свечу
3. Анализирует последние 2 часовых свечи
4. Если все свечи одного цвета - формирует сигнал
5. Отправляет сообщение в Telegram

## Особенности Finnhub API

- **Бесплатный тариф**: 60 запросов в минуту
- **Данные**: в реальном времени
- **Поддерживаемые пары**: EURUSD, GBPUSD, USDJPY и другие majors (с префиксом FX:)
- **Документация**: https://finnhub.io/docs/api
